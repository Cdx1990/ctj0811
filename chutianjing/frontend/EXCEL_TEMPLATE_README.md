# 数据资产质量分析 Excel 模板说明

## 概述

本系统提供了两种Excel导出功能，用于生成数据资产质量分析报告：

1. **完整版报告** - 包含6个工作表的详细分析报告
2. **简化版报告** - 包含1个工作表的基础数据报告

## 完整版报告结构

### 1. 概览工作表
包含以下内容：
- 报告基本信息（生成时间、统计周期）
- 关键指标概览（整体质量评分、数据总量、质量问题数、合规率）
- 质量等级分布统计
- 数据源质量排名

### 2. 详细数据工作表
包含所有数据资产的详细质量信息：
- 数据资产名称
- 质量评分
- 各维度得分（完整性、准确性、一致性、及时性、有效性）
- 问题数量
- 状态评级
- 最后更新时间

### 3. 质量维度分析工作表
分析各质量维度的表现：
- 各维度的平均得分、最高得分、最低得分
- 标准差分析
- 改进建议
- 维度权重配置

### 4. 问题分析工作表
详细分析质量问题：
- 问题类型分布（数据缺失、格式错误、重复数据等）
- 问题趋势分析（月度变化）
- 影响程度评估
- 解决方案建议

### 5. 趋势分析工作表
展示质量变化趋势：
- 月度质量评分变化
- 各维度趋势分析
- 预测分析（未来3个月）
- 影响因素分析

### 6. 改进建议工作表
提供具体的改进方案：
- 改进项目优先级排序
- 预期效果评估
- 实施周期和资源需求
- 风险控制措施

## 简化版报告结构

简化版报告包含一个工作表，主要展示：
- 数据资产列表
- 质量评分
- 各维度得分
- 问题数量
- 状态评级

## 使用方法

### 在页面中使用

1. 点击"导出完整报告"按钮生成包含6个工作表的详细报告
2. 点击"导出简化版"按钮生成包含1个工作表的基础报告

### 编程方式使用

```javascript
import { generateDataQualityReport, generateSimpleReport } from '../utils/excelTemplate.js'

// 准备数据
const data = {
  overallScore: 87.5,
  dataVolume: '2.3TB',
  issuesCount: 156,
  complianceRate: 94.2,
  tableData: [
    {
      name: '客户基础信息',
      score: 92,
      completeness: 95,
      accuracy: 89,
      consistency: 94,
      timeliness: 90,
      issues: 8,
      status: '优秀'
    }
    // ... 更多数据
  ]
}

// 生成完整版报告
generateDataQualityReport(data)

// 生成简化版报告
generateSimpleReport(data)
```

## 文件命名规则

- 完整版报告：`数据资产质量分析报告_YYYYMMDD.xlsx`
- 简化版报告：`数据质量报告_YYYYMMDD.xlsx`

## 技术特性

### 自动格式化
- 自动设置合适的列宽
- 数据格式化（百分比、日期等）
- 中文支持

### 数据完整性
- 包含所有必要的质量指标
- 提供多维度的分析视角
- 支持趋势分析和预测

### 可扩展性
- 模块化的模板设计
- 易于添加新的工作表
- 支持自定义数据格式

## 注意事项

1. **数据格式**：确保传入的数据格式正确，特别是百分比和数值类型
2. **文件大小**：完整版报告可能较大，建议在下载时显示进度提示
3. **浏览器兼容性**：支持所有现代浏览器，包括移动端
4. **数据安全**：Excel文件在客户端生成，不会上传到服务器

## 自定义配置

可以通过修改 `excelTemplate.js` 文件来自定义：

- 工作表名称
- 列宽设置
- 数据格式
- 文件命名规则
- 样式设置（如果支持）
