# 数据资产质量分析系统

## 📊 项目概述

这是一个基于Vue 3构建的数据资产质量分析动态看板系统，采用科技风格设计，提供完整的Excel下载功能。系统支持多维度数据质量分析，包括完整性、准确性、一致性、及时性和有效性等维度。

## ✨ 主要功能

### 🎯 核心功能
- **数据质量分析看板** - 实时展示数据资产质量状况
- **多维度质量评估** - 完整性、准确性、一致性、及时性、有效性
- **动态图表展示** - 趋势分析、维度分布、问题分析
- **Excel报告导出** - 支持完整版和简化版两种格式
- **科技风格界面** - 现代化的UI设计，优秀的用户体验

### 📈 分析维度
- **整体质量评分** - 综合质量指标
- **数据总量统计** - 数据规模监控
- **质量问题分析** - 问题类型和分布
- **合规率监控** - 数据合规性评估
- **趋势预测** - 质量变化趋势分析

## 🚀 技术栈

### 前端技术
- **Vue 3** - 渐进式JavaScript框架

## 📊 Excel模板功能

### 完整版报告 (6个工作表)
1. **概览** - 关键指标和统计信息
2. **详细数据** - 所有数据资产的质量详情
3. **维度分析** - 各质量维度的统计分析
4. **问题分析** - 质量问题分布和趋势
5. **趋势分析** - 质量变化趋势和预测
6. **改进建议** - 具体的改进方案和风险评估

### 简化版报告 (1个工作表)
- **数据质量报告** - 核心质量指标汇总

### 模板特性
- ✅ 自动格式化列宽
- ✅ 中文完全支持
- ✅ 日期时间戳
- ✅ 百分比格式
- ✅ 状态标识
- ✅ 多工作表支持

## 🛠️ 安装和运行

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd chutianjing
```

2. **安装依赖**
```bash
cd frontend
npm install
```

3. **启动开发服务器**
```bash
npm run dev
```

4. **访问应用**
```
http://localhost:5173
```

### 构建生产版本
```bash
npm run build
```

## 📖 使用说明

### 访问数据质量分析页面
1. 启动应用后，在侧边栏点击"📊 数据资产质量分析"
2. 页面将显示完整的数据质量分析看板

### 使用筛选功能
- **数据源筛选** - 选择特定的数据源
- **质量维度筛选** - 按维度类型筛选
- **时间范围筛选** - 选择分析时间范围
- **质量等级筛选** - 按质量等级筛选

### 导出Excel报告
- **导出完整报告** - 生成包含6个工作表的详细报告
- **导出简化版** - 生成包含1个工作表的基础报告

### 查看演示页面
访问 `demo.html` 可以体验Excel模板生成功能：
```bash
# 在浏览器中打开
open frontend/demo.html
```

## 🔧 自定义配置

### 修改Excel模板
编辑 `src/utils/excelTemplate.js` 文件：
- 修改工作表名称
- 调整列宽设置
- 自定义数据格式
- 更改文件命名规则

### 修改样式
编辑 `src/style.css` 文件：
- 调整颜色主题
- 修改布局样式
- 自定义动画效果

### 添加新的数据源
在 `DataQuality.vue` 中修改 `filters.dataSource` 选项：
```javascript
const filters = ref({
  dataSource: 'all',
  // 添加新的数据源选项
})
```

```javascript
{

    }
  ]
}
```
